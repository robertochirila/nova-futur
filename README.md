### Weather app developed using openweathermap API

### INSTALL DEPENDENCIES

npm install

### STRUCTURE

The app has three components:

-   App.js (statefull component) where we fetch the data and pass it down as props.
-   CurrentWeatherForecast.js (functional component with hooks) where we display the current weather and initiate the update function.
    -NextDaysWeatherForecast.js (functional component with hooks) where we display the weather forecast for the next days.

### DESIGN

The app follows a mobile-first design implemented using styled-components
